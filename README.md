# Air Quality Prediction (미세먼지 농도 예측)

📌 프로젝트 소개  
공공데이터 포털에서 제공하는 미세먼지 및 기상 관련 환경 데이터를 활용하여  
미세먼지 농도를 예측하는 머신러닝 모델을 설계한 프로젝트입니다.

칼럼 수가 많고 품질이 일정하지 않은 공공데이터를 직접 전처리하며,  
데이터 정제와 구조화가 모델 성능에 미치는 영향을 실습을 통해 경험했습니다.

---

🛠 사용 기술
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn

---

⚙️ 주요 기능
- 미세먼지 및 기상 공공데이터 수집 및 병합
- 결측치·이상치 처리 및 데이터 정제
- 학습용 데이터셋 구성
- 머신러닝 모델 학습 및 예측 결과 시각화

---

👩‍💻 담당 역할
- 공공데이터 수집 및 데이터 구조 분석
- 데이터 전처리(정제, 결측치 처리, 이상치 처리, 스케일링)
- 학습용 데이터셋 구성 및 모델 학습·평가
- 전처리 파이프라인 설계 및 시각화 수행

---

📈 결과
- 미세먼지 농도 예측을 위한 기본 머신러닝 모델 구축
- 전처리 방식에 따른 데이터 품질 및 예측 결과 차이 확인
- 공공데이터 기반 분석에서 전처리의 중요성 체감

---

📂 프로젝트 구성
- `공공데이터를 활용한 미세먼지 농도 예측.ipynb` :  
  데이터 전처리, 모델 학습 및 예측 수행
- `air_2024.csv`, `air_2025.csv` : 미세먼지 관련 데이터
- `weather_2024.csv`, `weather_2025.csv` : 기상 데이터
